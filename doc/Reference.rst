=========
Reference
=========

This section describes the layout of the code repository and contains
the API reference to the complete contents of the code collection.

Repository layout
=================

The repository contains the following directories:

:file:`scripts`
    Scripts in the code collection.

:file:`CGAT`
    Modules for code shared between scripts in the collection.

:file:`doc`
    The sphinx_ documentation of the code repository.

:file:`tests`
    Testing code

:file:`recipes`
    ipython_ notebook recipes and tutorials.

:file:`galaxy`
    scripts to hook the code collection into galaxy_.
